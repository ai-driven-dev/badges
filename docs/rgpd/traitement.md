# Information sur le traitement (RGPD, art. 13)

Notice à intégrer/référencer dans les CGU AIDD. **Information, pas consentement** :
la base légale est l'exécution du contrat (art. 6.1.b), le membre n'a rien à « accepter ».

| | |
|---|---|
| **Responsable** | AI-Driven Development |
| **Finalité** | Émettre et publier un badge de certification vérifiable, et l'annuaire public des membres certifiés. |
| **Base légale** | Exécution du contrat (art. 6.1.b) — la certification fait partie de l'adhésion. |
| **Données publiées** | Handle GitHub, nom, LinkedIn, photo, site et description (facultatifs), rôle, dates, identifiant de badge. Toutes **publiques par conception**. |
| **Destinataires** | Public (le badge et l'annuaire sont publics). Hébergement : GitHub. |
| **Durée** | Tant que la certification est active ; retirées sur demande de retrait. |
| **Droits** | Accès, rectification, effacement. L'**effacement** se fait par une demande de retrait (supprime fiche + photo, révoque le badge). Réclamation possible auprès de la CNIL. |

**Limite signalée** : une preuve déjà téléchargée par un tiers reste hors de notre
contrôle ; après retrait, elle apparaît **révoquée** à la vérification (seule voie).
Aucune donnée personnelle ne subsiste dans l'historique Git après retrait.
